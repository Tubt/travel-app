# Hướng dẫn Setup Tusk từ A-Z

## 🔑 Bước 1: Thêm Secret vào GitHub

1. **Vào GitHub repo của bạn** → Settings → Secrets and variables → Actions
2. **Click "New repository secret"**
3. **Điền thông tin**:
   - **Name**: `TUSK_AUTH_TOKEN`
   - **Secret**: `<your-tusk-auth-token>` (lấy từ Tusk dashboard)

## 🏢 Bước 2: Cấu hình Tusk (Trong Tusk Dashboard)

### Thông tin Repository:

```
Repository URL: https://github.com/<your-username>/<your-repo-name>
Branch: main (hoặc master)
Workflow File: .github/workflows/checklist.yml
```

### Thông tin Inputs (Tusk sẽ tự động điền):

```
runId: <auto-generated-by-tusk>
tuskUrl: https://your-tusk-instance.com
commitSha: <auto-detected-by-tusk>
```

## 📋 Bước 3: Kiểm tra Manual Test

Để test thủ công, vào GitHub repo → Actions → "Tusk Test Runner" → Run workflow:

```
runId: test-manual-123
tuskUrl: https://app.usetusk.ai
commitSha: <your-latest-commit-sha>
```

### Cách lấy commitSha:

```bash
git log --oneline -1
# Output: abc1234 Your latest commit message
# Dùng: abc1234
```

## 🌐 Bước 4: Tusk URL Examples

**Tusk Cloud (thông thường)**:

```
tuskUrl: https://app.usetusk.ai
```

**Tusk Self-hosted**:

```
tuskUrl: https://tusk.your-company.com
```

## 🔍 Bước 5: Auth Token - Lấy từ đâu?

1. **Login vào Tusk Dashboard**
2. **Vào Settings** → API Keys hoặc Integrations
3. **Tạo new token** cho GitHub integration
4. **Copy token** và paste vào GitHub Secrets

## ✅ Bước 6: Test Integration

### Manual Test từ GitHub:

1. Vào repo → Actions → "Tusk Test Runner" → "Run workflow"
2. Điền các thông tin test:
   ```
   runId: manual-test-001
   tuskUrl: https://app.usetusk.ai
   commitSha: <your-commit-sha>
   ```

### Kiểm tra kết quả:

- ✅ Workflow chạy thành công
- ✅ App container start (port 3000)
- ✅ Tusk runner thực thi tests
- ✅ Results xuất hiện trong Tusk dashboard

## 🐛 Troubleshooting

### Lỗi: "Invalid auth token"

**Fix**: Kiểm tra lại `TUSK_AUTH_TOKEN` secret

### Lỗi: "Workflow not found"

**Fix**: Đảm bảo file `.github/workflows/checklist.yml` có đúng format

### Lỗi: "App not responding"

**Fix**: Kiểm tra Docker service và port 3000

## 📝 Template cho Tusk Config

Nếu bạn setup từ Tusk dashboard, dùng config này:

```json
{
  "repository": "https://github.com/<username>/<repo>",
  "workflow": "checklist.yml",
  "branch": "main",
  "testFilePattern": "test/e2e/**/*.cy.ts",
  "environment": {
    "NODE_VERSION": "18"
  }
}
```
